

  Para utilizar el compilador, dado un archivo \textit{Ejemplo.willie}, se
ejecuta:

\begin{Verbatim}
> williec < Ejemplo.willie > Ejemplo.alf
\end{Verbatim}



El código de la máquina virtual está en el directorio
  /src/alfvm, para compilarlo se ejecuta:
\begin{verbatim}
  > cd src/alfvm
  > make
\end{verbatim}
